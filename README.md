# Dezolve Labs Site

Marketing website for Dezolve Labs built with Next.js App Router and optimized for Vercel deployment.

## Run locally

```bash
npm install
npm run dev
```

## Deploy

1. Push to GitHub.
2. Import the repository into Vercel.
3. Add `dezolvelabs.com` as a custom domain.
4. SSL and preview deployments are automatic on Vercel.
